# Claude Skills

Personal Claude skills for use across Claude Code, Claude.ai, and Cowork.

## Skills

### data-visualization

Tufte-inspired warm minimalist chart style. Applies to any visualization in R, Python, or other languages.

- Warm beige `#e5e1d8` backgrounds
- Earth-tone palette (`#5f3946` maroon, `#9c9280` grey-brown, `#3C3C3C` charcoal text)
- No gridlines, no legends (direct labeling), no chart junk
- Static output only (PNG/SVG)
- Includes ready-to-use R (ggplot2) and Python (matplotlib) templates

## Usage

### Claude Code
Copy skill files to `~/.claude/skills/`.

### Claude.ai / Claude App
Upload as a skill via Settings > Profile > Skills, or paste into project instructions.
